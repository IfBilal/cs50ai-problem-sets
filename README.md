## 📌 Overview

This repository contains my implementations of Harvard’s **CS50AI (Introduction to Artificial Intelligence with Python)**.

Each concept is not just studied, but implemented as a working system — ranging from classical search algorithms to neural networks and language processing.

---

## 🚀 Projects

### 🧩 Project 0 – Search

- **Degrees**
  - Finds shortest connection between actors using **Breadth-First Search (BFS)**
- **Tic-Tac-Toe**
  - AI agent using **Minimax algorithm**
  - Plays optimally against any opponent

---

### 🧠 Project 1 – Knowledge

- **Knights**
  - Solves logic puzzles using **propositional logic**
- **Minesweeper**
  - AI agent using:
    - Logical inference
    - Knowledge base construction

---

### 🎲 Project 2 – Uncertainty

- **PageRank**
  - Models ranking using **probability distributions**
- **Heredity**
  - Computes genetic inheritance using **Bayesian reasoning**

---

### 🎯 Project 3 – Optimization

- **Crossword**
  - Solves crossword puzzles using:
    - Constraint Satisfaction Problems (CSP)
    - Backtracking
    - AC-3 algorithm
    - Heuristics (MRV, Degree)

---

### 📊 Project 4 – Learning

- **Shopping**
  - Predicts user behavior using **classification models**
- **Nim**
  - Reinforcement learning agent using **Q-learning**

---

### 🧾 Project 5 – Neural Networks

- **Traffic**
  - Image classification using **Convolutional Neural Networks (CNNs)**

---

### 💬 Project 6 – Language

- **Parser**
  - Sentence parsing using **context-free grammar**
- **Attention**
  - Implements **attention mechanism** to analyze word importance

## 🧪 Running the Projects

```bash
git clone https://github.com/IfBilal/cs50ai-problem-sets.git
cd cs50ai-problem-sets
```

Navigate to any project:

```bash
cd project-folder
python3 file.py
```

Install dependencies if needed:

```bash
pip install -r requirements.txt
```
